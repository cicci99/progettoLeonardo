PROGETTO LEONARDO

CHI SIAMO?

Benvenuti, questa pagina è frutto di un progetto di alternanza scuola-lavoro in collaborazione con la società informatica Fides Consulting.
Lo scopo del progetto è istruire noi studenti all'utilizzo del linguaggio Html, Css e JavaScript per una corretta creazione e sviluppo di
un sito web dall'apertura del primo file fino alla messa in rete.
Durante il periodo di alternanza ci sono state mostrate le procedure e i siti di training ai sistemi di scripting e ci sono state assegnate
alcune attività per applicare e migliorare le capicità acquisite durante il training.
